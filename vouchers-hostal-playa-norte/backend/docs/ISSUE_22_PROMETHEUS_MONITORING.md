### Tipos de M√©tricas

#### 1. **Counter** (Contador)
```javascript

// Uso: totales, eventos acumulativos

```
- ‚úÖ Requests totales
- ‚úÖ Errores totales
- ‚úÖ Eventos creados

#### 2. **Gauge** (Medidor)
```javascript

// Uso: valores que fluct√∫an

gauge.inc();    // +1 conexi√≥n
gauge.dec();    // -1 conexi√≥n
```
- ‚úÖ Conexiones activas
- ‚úÖ Tama√±o de cach√©
- ‚úÖ Memoria utilizada

#### 3. **Histogram** (Distribuci√≥n)
```javascript

// Uso: latencias, tama√±os

```
- ‚úÖ Latencias HTTP
- ‚úÖ Tiempos de BD
- ‚úÖ Tama√±os de payload

#### 4. **Summary** (Resumen)
```javascript

// Uso: agregaciones r√°pidas

```
- ‚úÖ Estad√≠sticas de uso
- ‚úÖ M√©tricas de negocio

## ‚ö†Ô∏è Alerting Rules

#### 1. **HighErrorRate**

- **Condici√≥n:** >5% de errores en 5 minutos

- **Condici√≥n:** P99 latencia >1 segundo
- **Acci√≥n:** Investigar cuellos de botella

#### 3. **DatabaseSlow**
```promql

```
- **Condici√≥n:** P95 BD latencia >500ms
- **Acci√≥n:** Revisar √≠ndices, queries

#### 4. **CacheHighMissRate**
```promql

```
- **Condici√≥n:** >30% miss rate
- **Acci√≥n:** Revisar estrategia de cach√©

### Agregar Alertas Personalizadas

**alertmanager.yml:**
```yaml

  - name: hostal_alerts
    interval: 30s
    rules:
      # HTTP Errors
```

## üîß Troubleshooting

### Problema: `/metrics` devuelve 404

**Causa:** Middleware no registrado  
**Soluci√≥n:**
```javascript

```

### Problema: M√©tricas no se incrementan

**Causa:** Las m√©tricas se registran pero no se usan  
**Verificar:**
```javascript
// ‚úÖ Correcto

// ‚ùå Incorrecto - no incrementa
counter.get();
```

### Problema: Memoria crece indefinidamente

**Causa:** Labels no acotados  
**Soluci√≥n:** Limitar valores de labels
```javascript

counter.inc(1, { path: req.path });

// ‚úÖ Bien - valores predefinidos
const endpoint = normalizeEndpoint(req.path);
counter.inc(1, { endpoint });
```

### Problema: Prometheus no puede scrape

**Verificar:**
```bash
# ¬øEl endpoint es accesible?
curl http://localhost:3000/metrics

# ¬øEl formato es v√°lido?
# Debe tener l√≠neas "# HELP" y "# TYPE"

# ¬øPrometheus tiene red al endpoint?
docker exec prometheus_container curl http://api:3000/metrics
```

### Problema: Alertas no disparan

**Verificar:**
1. Alert rules est√°n en prometheus.yml
2. Expresi√≥n PromQL es v√°lida
3. `for:` es correcto (tiempo m√≠nimo)

```promql
# Probar en Prometheus UI (http://localhost:9090)
rate(hostal_api_http_errors_total[5m]) > 0.05
```

### Problema: Alto uso de CPU por Prometheus

**Causas:**
- Scrape interval muy bajo
- Demasiadas m√©tricas
- Queries muy complejas

**Soluciones:**
```yaml
# prometheus.yml
scrape_configs:
  - job_name: 'hostal'
    scrape_interval: 30s  # Aumentar
    scrape_timeout: 10s
```
